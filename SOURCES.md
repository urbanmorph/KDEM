## VERIFICATION PROTOCOL

### For New Data Addition:
1. Identify primary source (official > research > news)
2. Cross-validate with 2+ sources where possible
3. Assign confidence rating per framework
4. Document methodology for estimates
5. Note verification date
6. Flag for periodic re-verification

### For Dashboard Publication:
1. All Level 1-2 confidence data: Publish directly
2. Level 3 data: Include source citation
3. Level 4-5 data: Include "Reported" or "Estimated" tag
4. Aspirational targets: Clear "Target" label
